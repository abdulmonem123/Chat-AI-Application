# 🤖 Chat AI Application

A powerful and modern **Chat AI Application** built using **Hugging Face SmolAgents** and powered by the **Kimi-K2.5 (moonshotai) API model**. This app provides intelligent, context-aware conversations with a fast and responsive user interface.

---

## 🚀 Features

- 💬 Real-time AI chat  
- 🧠 Context-aware intelligent responses  
- 🤖 Agent-based system using SmolAgents  
- ⚡ Fast API integration with Kimi-K2.5  
- 🎨 Clean UI built with React & Material UI  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Material UI  
- JavaScript  

### Backend
- Python  
- FastAPI / Flask  
- Hugging Face SmolAgents  

### AI Model
- moonshotai/Kimi-K2.5
